## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/Hemanth1310/home-decor-ecommerce.git]
   ```
2. Navigate to the project directory:
   ```bash
   cd .
   ```

### Running the App

1. Open `index.html` in your preferred browser or use a local server
